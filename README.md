# Launch Express App

A quick and minimalistic way to get started with express.js server-side development 

## Quick Overview

Follow this command to get started

```bash
  npx launch-express-app my-app
  cd my-app
  npm install
```

## Running Server

To run server, run the following command

```bash
  npm run start-dev
```
